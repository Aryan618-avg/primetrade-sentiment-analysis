# BTC Sentiment × Trader Performance Analysis
### PrimetradeAI Internship Assignment — Data Science Role

## Overview
Exploratory analysis of the relationship between Bitcoin market sentiment 
(Fear & Greed Index) and trader performance on Hyperliquid exchange.

## Datasets Used
| Dataset | Source | Size |
|---|---|---|
| Bitcoin Fear & Greed Index | Provided | 2,644 rows |
| Hyperliquid Historical Trades | Provided | 211,224 rows |

## Key Findings
- **83.2% overall win rate** across 104,402 closed positions
- **Fear = Best time to BUY** → avg $209/trade vs $28 in Extreme Greed
- **Extreme Greed = Best time to SELL** → avg $176/trade
- **Correlation of −0.098** between daily PnL and sentiment score
- One trader achieved **99.1% win rate** across ~10,000 trades
- **@107, HYPE, SOL** generated 62% of all profits
